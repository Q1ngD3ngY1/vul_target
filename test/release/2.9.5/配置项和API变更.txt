----------七彩石----------


1. 新增数据库发布任务的异步任务

application.yaml -> tasks
  enable_db_source_task:
    runners: 3 #处理协程数，无法动态调整
    retry_wait_time: 1m #重试等待时间
    max_retry: 2 #最大重试次数
    timeout: 10m #超时时间
    fail_timeout: 3s # 失败回调超时
    delay: 0s #启动延迟
    batch: 1
    batch_size: 0
    stopped_resume_time: 720h
在main.yaml中的 handle_tasks 中增加
  - "enable_db_source_task"

2. 新增数据表学习异步任务

application.yaml -> tasks
  add_db_table_task:

在main.yaml中的 handle_tasks 中增加

3. 新增外部数据库控制参数，新增 max_top_value_row， top_value_num 修改 es_input_size 为 1000， 新增数据库 VDB 控制开关
db_source:
  es_input_size: 1000
  # 获取 top value 的最大行数
  max_top_value_row: 200000
  # 获取 top value 的数量
  top_value_num: 5
  # 是否启用数据库 VDB 功能
  enable_vdb: false
  # 允许添加的最大数据库数量
  max_db_source_num: 100
